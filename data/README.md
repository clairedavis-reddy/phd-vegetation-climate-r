# Sample data

This folder contains lightweight example datasets included so the public notebooks run out of the box.

- `climate_timeseries_sample.csv` — synthetic annual temperature and rainfall series, structured like a gridded climate summary.
- `ndvi_monthly_sample.csv` — synthetic monthly NDVI and rainfall series for one site.
- `vegetation_grid_sample.csv` — synthetic gridded vegetation table used to demonstrate pixel-wise trend estimation without shipping large rasters.

These files are illustrative only. They preserve the logic of the original workflows without exposing private or licensed research data.
